## VueJS Simple Food Menu Recipe
VueJS Simple Food Menu Recipe

## Requirements
1. NodeJS
2. yarn or npm

## Installation 
1. Clone repo `git clone git@github.com:fahmipotlot/vuejs-recipe-menu.git`
2. Copy .env file `cp .env.example .env`
3. Set Up VITE_API_ENDPOINT in .env
4. Run `yarn install`
5. Run `yarn run dev`
